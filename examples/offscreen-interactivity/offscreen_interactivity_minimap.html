Offscreen Interactivity - External Minimap

📋 Two menus to explore

  • This example shows how to use the Minimap component in an external container. The minimap is not part of the editor's DOM tree, but it is still interactive and synchronized with the editor.
  • The minimap is created with the Minimap class, which takes the editor instance and a configuration object as parameters. The minimap is then attached to an external container using the attach method. The minimap will automatically update when the editor's content changes, and it will also respond to user interactions such as zooming and panning.
