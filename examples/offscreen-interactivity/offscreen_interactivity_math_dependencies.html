Math Dependencies Example

🔗 What are Math Dependencies?

When you write math expressions that use variables, the system tracks which blocks depend on which variables.

  • Source blocks define variables (e.g., "x=2")
  • Dependent blocks use those variables (e.g., "3x+2=")
  • When a variable changes, dependent expressions recalculate automatically
  • Hover over blocks to see dependencies highlighted with colored arrows
📝 Try it:
1. Write a variable definition like x=2
2. Write an expression using that variable like 3x+2=
3. The result computes automatically when you end with = or ? (if Auto-compute is enabled)
4. Toggle "Auto-compute" to switch between automatic and manual computation
5. Change the variable value using "Edit Variables"
6. Watch dependent expressions recalculate in real-time
7. Hover over math blocks to see dependencies highlighted (sources in green, dependents in orange)
