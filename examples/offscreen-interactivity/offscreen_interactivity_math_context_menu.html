Math Context Menu Example

📋 How to use the Math Context Menu

The Math context menu provides powerful operations for mathematical expressions. To access it:

  1. Draw a circle or rectangle around the expression you want to work with (this selects it)
  2. Choose an operation from the menu

✏️ Try these examples:

  1. Computation: Write √16=, draw around it, Compute numerical result
  2. Variable substitution: Write x=3, then 2x+5=, draw around the second expression, Compute numerical result (will use x=3 value)
  3. Function plot: Write f(x)=x², draw around it, Evaluate function → Set range from -5 to 5
  4. Reversible function plot: Write y=x², draw around it, Evaluate function → you can choose to plot y = f(x) or x = f(y)
  5. Complex expression: Write a=2, then 3a²+2a-1=, draw around the expression, Edit variables → Change a value, Compute numerical result (will use the updated value)

💡 Tips

  • End expressions with = or ? to indicate you want a result
  • Variables are automatically detected when you define them (e.g., x=5)
  • You can use the same variable in multiple expressions
  • The solver supports common math functions: sin, cos, tan, sqrt, exp, log, etc.
  • For evaluation, the function must have a clear input-output relationship (e.g., y=f(x))
