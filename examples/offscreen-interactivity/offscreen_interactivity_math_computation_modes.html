Math Computation Modes

🎯 What are computation modes?

When solving a math expression, the result can be displayed in two different ways:

✏️ Draw Result

  • Result added as real ink strokes in the document
  • Sent to backend — recognized and part of the model
  • Fully interactive: move, erase, select
  • With auto-compute: result strokes automatically cleared and recomputed when the expression label changes
  • Result strokes also cleared when a dependent block changes (e.g. x=2 changes → 2x= result cleared)
  • To remove manually: use Math menu → Select result strokes → Delete result strokes

👻 Show Result (Ghost)

  • Result shown as SVG overlay at 50% opacity
  • Not in model, not sent to backend
  • Non-interactive — view only
  • With auto-compute: refreshed on every sync. Ghost cleared and recomputed when expression changes
  • Automatically cleared when the block no longer ends with = or ?

⚙️ Auto-Compute

Automatically compute expressions ending with = or ? after each recognition sync.

Both modes: when a block's recognized expression changes (label change), existing solver output strokes are automatically cleared — for the block itself and any dependent blocks. A fresh computation follows immediately.

Ghost mode: result re-rendered on every sync. Cleared automatically when block no longer ends with = or ?.

Draw mode: result stays as ink when expression is stable. Cleared and recomputed only when expression changes. Use the Math menu to manage result strokes manually.

// Ghost mode — safe for auto-compute editor.math.updateComputationConfig({ resultMode: "ghost", autoCompute: true }) // Draw mode — result persists, computed once per block editor.math.updateComputationConfig({ resultMode: "draw", autoCompute: true })
📝 Try it:

Ghost mode:
1. Select "Show result" + enable "Auto-compute"
2. Write 5+3= → ghost result appears immediately
3. Modify the expression → ghost updates in real time
4. Remove the = → ghost disappears automatically

Draw mode:
1. Select "Draw result" + enable "Auto-compute"
2. Write 5+3= → result added as ink strokes
3. Modify the expression (e.g. change 3 to 30) → old result cleared, new result computed
4. Write x=2 then 2x= → both computed
5. Change x=2 to x=5 → 2x= result automatically cleared and updated to 10
6. Use Math menu → Select result strokes then Delete result strokes to remove manually
