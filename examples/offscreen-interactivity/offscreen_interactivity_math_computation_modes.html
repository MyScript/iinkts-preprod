Math Computation Modes

🎯 What are computation modes?

When solving a math expression, the result can be displayed in two different ways:

✏️ Draw Result

  • Result rendered as actual strokes
  • Sent to backend (recognized ink)
  • Becomes part of document
  • Fully interactive (move, erase, etc.)

👻 Show Result (Ghost)

  • Result shown with 50% opacity
  • Not sent to backend
  • Visual preview only
  • Non-interactive (view-only)

⚙️ Auto-Compute

Automatically compute expressions when you finish writing them (end with = or ?).

How it works: After writing a block, synchronized event triggers auto-compute check. If block ends with = or ?, solver computes result.

Why useful: Real-time feedback without manual "Compute" clicks.

📝 Try it:
1. Write 5+3= (with equals sign)
2. If auto-compute enabled → result appears instantly
3. Toggle "Auto-compute" checkbox to see difference
4. Switch "Result mode" to see Draw vs Ghost rendering
